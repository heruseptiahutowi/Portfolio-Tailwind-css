ID/X Partners Data Engineer Project Based Internship Program at ID/X Partners

This is the final project of the virtual internship program held by Rakamin for data engineers at the ID/X Partners company. The task given was to help one of the clients from the ID/X Partners company regarding creating a data warehouse. The problem they experienced was that they had difficulty extracting data from various sources and data types into one data warehouse. The process of working on this project starts from creating a data warehouse called DWH using the Microsoft SQL Server database application. The creation of this data warehouse refers to the needs according to the client's orders. After creating the data warehouse, data from various sources and data types is transformed using the Talend Studio application and at the final stage the data that has been transformed to the DWH data warehouse is created as a stored producer called Daily Transaction and BalancePerCustomer. The process of working on this project was very impressive starting from the extract, transfer and load stages for me because it was very insightful
